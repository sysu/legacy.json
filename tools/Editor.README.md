# editor.py

基于 tkinter 的 legacy.json 图形化编辑器，用于创建和编辑 legacy.json 文件。

### 功能特性

- **文件操作**: 新建、打开、保存、另存为
- **字段编辑**: 左侧字段列表，支持多种输入类型
- **自动时间戳**: `update_time` 字段在保存时自动生成
- **多种字段类型支持**:
  - 单行文本
  - 多行文本
  - 字符串数组
  - 时间范围数组
  - 时间多行数组
  - 多行键值数组

### 运行环境

- Python 3.x
- tkinter（通常随 Python 一起安装）

### 使用方法

```bash
python editor.py
```

### 界面说明

- **工具栏**: 顶部工具栏包含文件操作按钮
- **左侧面板**: 字段列表，点击选择要编辑的字段
- **右侧面板**: 字段编辑区域，显示字段说明和输入组件
- **状态栏**: 底部显示当前文件状态

### 字段类型

| 类型     | 说明                            |
| ------ | ----------------------------- |
| 单行文本   | 简单文本输入，如 id、commit\_sha       |
| 只读文本   | 自动生成，如 update\_time           |
| 多行文本   | 大段文字，如 planning、description   |
| 字符串数组  | 多个字符串，如 names、keywords        |
| 时间范围数组 | 开始和结束时间，如 available\_times    |
| 时间多行数组 | 时间+描述，如 milestones            |
| 多行键值数组 | 标题+内容，如 resources、influencers |

### 注意事项

- 撤销/重做功能尚未实现
- 建议定期保存文件以防数据丢失
